# MX Ink WebXR

This project combines multiple challenges from LauzHack of EPFL, Switzerland, which are proposed by companies such as AXA Group, Logitech and primarily Vitol This project combines the power of Three.js and WebXR to create a versatile drawing tool that allows users to create three-dimensional artwork using VR controllers, complete with an innovative voice-activated AI assistant that can guide and enhance your creative process.

The integration of spatial audio and an AI assistant makes the experience more intuitive and interactive, allowing users to receive guidance, make adjustments, and explore new creative possibilities through natural voice commands while working on their 3D masterpieces.
